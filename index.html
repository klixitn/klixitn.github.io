Tentang

KLIX adalah sebuah komunitas yang bergerak di bidang Teknologi Informasi, khususnya penggunaan open source. KLIX berada dibawah naungan Teknik Elektro S-1 Intitut Teknologi Nasional Malang. KLIX memiliki basecamp di Gedung Elektro:

  • Laboratorium Jaringan Komputer & CISCO
  • Laboratoriom Algoritma dan Pemrograman

Komunitas Linux (KLIX) ITN Malang memiliki empat devisi, yaitu :

  • PEMROGRAMAN
  • WEB
  • JARINGAN
  • SECURITY

Pemrograman

Dalam Divisi Pemrograman Kita Belajar Tentang Dasar-Dasar Bahasa Pemrograman seperti python, C++ ,C , dsb. Serta segala sesuatu yang berhubungan dengan program.

Web

Dalam Divisi web Kita Belajar Mengenai Html, CSS, PHP, JavaScript, Dsb.

Jaringan

Dalam Divisi Jaringan Kita Belajar Mengenai Dasar-Dasar Jaringan, seperti: Pengenalang IP Address, VLSM, Routing, dsb.

Security

Dalam Divisi Security Kita Belajar Mengenai Autentikasi, Enskripsi, Bagaimana mengamankan jaringan maupun sistem jaringan dan komputer.
